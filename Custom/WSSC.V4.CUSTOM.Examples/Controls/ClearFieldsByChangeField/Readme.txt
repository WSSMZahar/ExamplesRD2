Постановка.
Контрол, который очищает поле типа подстановка (Поле2) при выполнении условий: 
1) в другом поле типа подстановка (Поле1) изменили значение (например, была указана Компания Альфа, ее изменили на компанию ВСС Докс или очистили поле); 
2) Поле2 - не пусто.

Названия полей 1 и 2 вынести в настройку.

Настройка.
1. Добавить контрол к списку:
Расположение: Очищение нескольких полей при изменении ключевого поля
Статус на форме: Редактирования
Класс контрола: WSSC.V4.DMS.FOS.Controls.ClearFieldsByChangeField.ClearFieldsByChangeField

2. В константах системы добавить элемент с системным названием «FOS_ ClearFieldsByChangeField»

Значение:

<Setting>
  <List ListName="Outgoing" WebUrl="dms/outgoing" >
    <Field Parent="Приоритет" Childs="Содержание;Подписант" />
    <Field Parent="Содержание" Childs="Вид документа" />
  </List>
</Setting>  
Где xml-узел «List» это настройка для списка, в котором содержит атрибут name – название списка; атрибут web это db-узел.
Xml-узел Field это настройка полей, где атрибут parent – ключевое поле, при изменении которого будут очищаться поля указанные через точка-запятую в атрибуте childs
